## Oppsett


### Backend
API-et er skrevet i C# i .NET Core. Åpne opp API-et i ditt foretrukne C#-IDE og kjør skiten. Den skal kjøre på localhost:5001

### Frontend
Gå i `frontend/src` og kjør `npm install` og `npm start`. Den kjører på localhost:5000.

### Proxy
Lokalt trenger vi også å kjøre en proxy. Den kjøres ved å gå i `frontend/src` og kjøre `node server.js`.
