# Bekk bemanning app

## Installation
Install yarn if you don`t have it 'npm -g yarn'
Install dependencies with `yarn` from the `src/` folder.

## Running
To run the project, run both `npm start` and `node server.js` in two separate console windows.

- `npm start` starts a webpack dev server which compiles, watches and reloads all project files.
- `server.js` starts a lightweight node web server that serves static files and hosts a config endpoint.

Now go to http://localhost:5000 to browse the application
